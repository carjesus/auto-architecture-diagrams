## [1.0.0] - 2024-01-12

### Added
- Initial release of Auto Architecture Diagrams
- Automatic repository analysis with `analyze_repo.py`
- AI-powered relationship refinement with `ai_refiner.py`
- Automatic architecture code generation with `generate_architecture.py`
- GitHub Actions workflows for manual and automatic diagram generation
- Support for multiple frameworks: FastAPI, Django, Flask, Express.js
- Support for multiple databases: PostgreSQL, MongoDB, MySQL, SQLite, Redis
- Support for cloud services: AWS (S3, RDS, Lambda, DynamoDB, SNS, SQS), GCP, Azure
- Comprehensive documentation and setup guides
- Example configurations and sample outputs
- Test scripts for validation
- MIT License
